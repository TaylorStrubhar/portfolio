Fictional Guacamole

This is a group project I worked on. It is an application for a boarding facility that users can input their pet's information and keep it on file for their reference and for the facility to access. This application utilizes Node, bcrypt, connect-session-sequalize, dotenv, express, express-handlebars, express-session, mysql2, seed and sequelize node packages. We also utilized Bulma as our CSS package and deployed the app using Heroku and JawsDB MySQL.
